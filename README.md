# Weight Prediction

Comparison and prediction of multiple models of my personal weight throughout the diet

### The following models are used:

- Linear Regression
- Random Forest
- Ridge Regression
- ARIMAX
- XGBoost

### Limitations and Future Plans:

- Works only for the next 7 days (43rd to 50th day). Update and verify how well each model performed (Continuous)
- Also update the number of days to predict
- Use Seaborn for other representations (One Time)
- Add other models like Prophet after better knowledge about forecasting
